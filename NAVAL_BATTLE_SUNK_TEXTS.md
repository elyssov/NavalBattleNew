# Naval Battle — Тексты гибели кораблей

## Задача

Заменить массивы `SUNK_DESCRIPTIONS` и `CAPTAIN_LAST_WORDS` в index.html.
Также убрать дублирование текста «Цель уничтожена» в строках, где `sunkDescription()` уже содержит полное описание гибели.

---

## CAPTAIN_LAST_WORDS

Массив строк. Шаблон `{NAME}` заменяется на имя корабля при вызове.

Старые 10 записей сохраняются, добавляются 5 новых (номера 11-15).

```
'«{NAME}»: "Я первый помощник, Семёнов. Капитан погиб. Принимаю командование на с—" ...белый шум. Тишина.'

'«{NAME}», мичман из БЧ-5: "Вода в третьем... четвёртом... переборка между ними сложилась как бумага... ребята не успели выйти..." Конец связи.'

'«{NAME}»: "Пожарным расчётам — отставить. Боезапас не потушить. Всем наверх. Все наверх, мать вашу!" ...хрип... молчание.'

'«{NAME}», последний перехват: "...ватерлиния разрушена по левому борту на протяжении тридцати метров... крен нарастает... контрзатопление невозможно — насосы обесточены..." Обрыв.'

'«{NAME}» — из радиорубки, открытым текстом, без шифра: "Корабль гибнет. Координаты 57-14. Флаг на месте. Экипаж... экипаж выполнил долг."'
```

## Правки в вызовах addLog

Убрать дублирование «Цель уничтожена» / «наблюдаю огонь по всей длине корпуса» после `sunkDescription()`, потому что описания гибели теперь самодостаточные.

**Строка с артиллерией (примерно ~3071):**
Было: `sunkType + ' противника поражен! ' + sunkDescription() + ' Цель уничтожена!'`
Стало: `sunkType + ' противника поражен! ' + sunkDescription()`

**Строка с ПКР (примерно ~3966):**
Было: `'ПКР прорвалась через ПВО! ' + tType + ' — ПРЯМОЕ ПОПАДАНИЕ! ' + sunkDescription() + ' Цель уничтожена, наблюдаю огонь по всей длине корпуса!'`
Стало: `'ПКР прорвалась через ПВО! ' + tType + ' — ПРЯМОЕ ПОПАДАНИЕ! ' + sunkDescription()`

Остальные вызовы `sunkDescription()` (АСРОК, торпеды, авиация) — без хвостов, оставить как есть.
