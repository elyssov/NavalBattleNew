# Naval Battle: Strategic Edition / Морской Бой: Стратегическая Версия

A turn-based naval strategy game — advanced Battleship with fog of war, recon mechanics, nuclear weapons, and diverse ship classes. Built with vanilla JS, HTML5 Canvas, no frameworks.

Пошаговая морская стратегия — продвинутый «Морской Бой» с туманом войны, разведкой, ядерным оружием и разнообразными классами кораблей. Написана на чистом JS, HTML5 Canvas, без фреймворков.

## Play Online

https://elyssov.github.io/NavalBattleNew/

## Features

- 3 field sizes (30×30, 50×50, 100×100) with 4 landscape types
- 7 ship classes from patrol boats to nuclear cruisers and aircraft carriers
- Radar scanning, air reconnaissance with detection aging
- PCR "Vulkan" missiles, torpedoes, striker aircraft, nuclear strikes
- Reactor meltdown and chain detonation mechanics
